## üöÄ Getting Started

### Prerequisites

- JDK 17 or higher
- Maven 3.6+
- Your favorite IDE (IntelliJ IDEA recommended)

### Running the Project

```bash
# Clone the repository
git clone <repository-url>

# Navigate to a specific chapter module
cd chapter-<number>

# Build the project
mvn clean install

# Run the Spring Boot application
mvn spring-boot:run
```
